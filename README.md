# Teal Team Cleaning

Teal Team Cleaning website — Greenville, SC.

## Setup

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub and connect to Vercel. Vercel will auto-detect Vite.

## Images

Place the following images in the `/public` folder before deploying:

- `hero.jpg` — bright kitchen/interior shot (hero background)
- `cleaning.jpg` — cleaning action shot (How It Works section)
- `supplies.jpg` — cleaning supplies shot (About section)
- `duster.jpg` — duster/detail shot (optional accent)

## Config

- EmailJS Service ID: `service_l19rftg`
- EmailJS Template: `template_tealteam` (create this in EmailJS dashboard)
- Booking Koala URL: replace the `alert()` in the book function with your URL
